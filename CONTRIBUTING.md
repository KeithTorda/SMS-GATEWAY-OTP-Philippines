# Contributing to SMS Gateway Philippines

First off, thank you for considering contributing to this project! It's people like you that make the open source community such a great place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
- Use the [Issue Tracker](https://github.com/KeithTorda/SMS-GATEWAY-OTP-Philippines/issues) to report bugs.
- Include details about your environment (PHP version, Android version).
- Provide a clear description of the problem and steps to reproduce it.

### Suggesting Enhancements
- If you have an idea to improve the system, feel free to open an issue with the tag "enhancement".
- Explain why this feature would be useful to most users.

### Pull Requests
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/your-feature`).
3. Commit your changes with descriptive messages.
4. Push to the branch (`git push origin feature/your-feature`).
5. Open a Pull Request.

## Coding Standards
- Maintain the MVC structure of the project.
- Ensure all sensitive data continues to use `.env` configuration.
- Keep the UI clean and responsive.
